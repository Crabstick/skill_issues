gitbook

使用gitbook写书 https://github.com/GitbookIO/gitbook

安装 npm install gitbook -g

例如 github上有项目 https://github.com/duoduo369/skill_issues 需要有SUMMARY.md and README.md

gitbook build -g duoduo369/skill_issues
gitbook serve -g duoduo369/skill_issues

SUMMARY格式 章节之间不能有空格
